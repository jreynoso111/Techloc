Cómo usar este panel

  1. Usa Recargar CSV para leer los datos publicados en /assets.
  2. Haz clic en cualquier celda para editarla o pulsa Agregar fila para sumar registros.
  3. Con Reemplazar CSV puedes cargar un archivo local y continuar editándolo.
  4. Cuando termines, pulsa Exportar y reemplaza el archivo original con el descargado.

Próximos pasos

Protege la carpeta /admin en tu hosting y envía estos CSV a la nube cuando migres a backend. Este panel seguirá funcionando sin cambios.
